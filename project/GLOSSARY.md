# Glossary — The Drake Display

A comprehensive reference of scientific terminology, proper names, and code libraries used across the project's research library, model specification, planning documents, and schemas. Compiled from every Markdown document in the repository. Organized into three parts: **scientific terms of art**, **proper names**, and **code libraries & data formats**.

---

## 1. Scientific & Technical Terms of Art

### 1.1 Mechanics, Rotation & Structural Physics

- **Angular momentum** — conserved rotational quantity; reorienting a megastructure demands enormous torque and time.
- **Angular velocity (ω)** — rotation rate; drives the centrifugal pseudo-acceleration formula ω²·r.
- **Artificial / spin gravity** — simulated gravity produced by rotation; physiologically similar to, but not identical to, true gravity.
- **Centrifugal force** — a fictitious outward pseudo-force that appears only when doing physics bookkeeping in a rotating (non-inertial) reference frame.
- **Centripetal force** — the real force causing curved motion, always directed toward the center of curvature (F = mv²/r).
- **Coriolis effect / force** — fictitious force (2m(ω × v)) acting on anything moving relative to a rotating structure; a source of disorientation in spin habitats.
- **Cross-coupled semicircular canal stimulation** — inner-ear mechanism causing nausea/disorientation when the head moves during rotation.
- **Hoop stress (σ = ρv² = ρgr)** — structural stress in a spinning shell or ring, scaling linearly with radius at fixed simulated gravity.
- **Megastructure** — a very large engineered astronomical structure (e.g., Dyson sphere, ringworld, stellar engine).
- **Neutral stability** — a configuration (e.g., a shell around a star) with no restoring force to correct displacement, so perturbations grow.
- **Non-inertial reference frame** — a rotating (accelerating) frame in which fictitious forces like centrifugal and Coriolis must be invented to make Newton's laws work.
- **Normal force** — the reaction push (e.g., a habitat hull on an occupant) felt as "weight" in a spin habitat.
- **Reactive centrifugal force** — the real Newton's-third-law partner of centripetal force, acting on the source of the centripetal force, not the orbiting mass.
- **Rim (tangential) velocity** — v = √(gr); the speed of a rotating habitat's outer edge.
- **Shell theorem** — a rigid spherical shell has zero net gravitational effect on anything inside it.
- **Specific strength** — a strength-to-weight material property; relevant to whether any known material can span a Dyson-sphere-scale structure.
- **Stellar engine** — a ring-shaped megastructure engineered around a star.
- **Tensile strength** — material property limiting how large a rotating structure or star-enclosing shell can be.

### 1.2 Cosmology

- **3-torus topology** — a hypothetical finite-but-unbounded universe shape.
- **Alcubierre-style metric engineering** — exotic-matter-based FTL-adjacent spacetime metric concept.
- **Anthropic / multiverse selection** — explanation that only universes with a tiny cosmological constant permit observers to exist and ask the question.
- **Arrow of time** — the puzzle of why time has a preferred direction despite time-symmetric microscopic laws.
- **Axis of Evil** — apparent anomalous alignment of large-scale CMB multipoles.
- **Baryon acoustic oscillations (BAO)** — a large-scale structure feature used with CMB and supernova data to test dark-energy models.
- **Baryon asymmetry** — the tiny leftover of matter over antimatter after the Big Bang.
- **Big Rip** — a hypothetical future in which phantom dark energy tears the universe apart.
- **Bouncing / cyclic cosmology** — an alternative to inflation involving a prior contracting phase.
- **Cold Spot** — an anomalous cold region in the CMB.
- **Cosmic inflation** — a brief epoch of exponential early-universe expansion.
- **Cosmic microwave background (CMB)** — relic radiation used to calibrate H₀ and probe inflation, isotropy, and cosmic topology.
- **Cosmological constant (Λ)** — the fixed vacuum energy density in the default ΛCDM dark-energy model.
- **Cosmological constant problem / vacuum catastrophe** — the ~10¹²⁰-factor mismatch between predicted and observed vacuum energy density.
- **CPL parametrization** — a specific fitting function for evolving dark energy, flagged as a possible source of a false "evolving dark energy" signal.
- **Dark energy** — the repulsive energy component (~68% of the cosmic energy budget) driving accelerating cosmic expansion.
- **Dark matter** — non-luminous gravitating matter (~27% of the universe), evidenced by rotation curves, lensing, and CMB peaks.
- **Distance ladder** — the Cepheid-calibrated method for measuring the local Hubble constant.
- **Ekpyrotic model** — a cyclic-cosmology alternative to inflation.
- **Electroweak baryogenesis** — an alternative baryon-asymmetry mechanism requiring new electroweak-scale physics.
- **Eternal inflation** — the hypothesis that inflation never fully stops, spawning a multiverse of bubble universes.
- **Flat universe / geometry** — cosmic geometry, measured flat to ~0.4%.
- **Galaxy rotation curves** — evidence for dark matter from anomalous orbital velocities.
- **Gravitational lensing** — the bending of light by mass, used as dark-matter evidence and as a megastructure/black-hole search technique.
- **Hartle–Hawking "no-boundary" proposal** — a quantum-gravity model in which asking what came "before" the Big Bang is meaningless.
- **Heat death** — ΛCDM's default long-term cosmic fate of eternal expansion.
- **Hubble constant (H₀)** — the universe's current expansion rate.
- **Hubble tension** — the >5σ disagreement between local (Cepheid/SNe Ia) and CMB-derived measurements of H₀.
- **Inflaton** — a hypothetical scalar field driving cosmic inflation.
- **ΛCDM** — the standard cosmological model combining a cosmological constant and cold dark matter.
- **Leptogenesis** — the leading hypothesis that baryon asymmetry originated in the neutrino sector via heavy Majorana neutrinos.
- **Loop quantum cosmology** — a quantum-gravity model replacing the Big Bang singularity with a "bounce."
- **Matched circles** — a CMB search technique for a wrapped-around (finite) cosmic topology.
- **Metric expansion of space** — the physically correct description of cosmic expansion: space itself expanding, not an explosion in space.
- **Modified gravity** — a proposed alternative to dark energy in which general relativity itself fails at large scales.
- **Modified Newtonian Dynamics (MOND)** — a modified-gravity alternative to dark matter that fits galaxy scales but fails at cluster scales.
- **Past Hypothesis** — the postulate that the early universe simply began in a low-entropy state.
- **Phantom dark energy** — an exotic dark-energy form that can violate energy conditions, potentially leading to a Big Rip.
- **Primordial black holes** — a dark-matter candidate formed in the early universe.
- **Quantum gravity** — the unknown physics needed to describe the Big Bang and singularities.
- **Quintessence** — dynamical scalar-field dark energy whose density evolves over time.
- **S₈/σ₈ tension (structure-growth tension)** — a second cosmological tension possibly linked to the Hubble tension.
- **Sakharov conditions** — the three requirements (baryon-number violation, C/CP violation, thermal non-equilibrium) needed to generate baryon asymmetry.
- **Singularity** — a point of infinite density where general relativity breaks down, e.g. at the Big Bang.
- **Sterile neutrinos** — a hypothetical non-interacting neutrino, proposed as a dark-matter candidate.
- **Strong-CP problem** — a particle-physics puzzle that axions are proposed to solve.
- **Vacuum decay** — the hypothesis that our cosmic vacuum is metastable and could catastrophically "decay."
- **WIMPs (weakly interacting massive particles)** — a long-favored but increasingly disfavored dark-matter candidate.
- **Axions** — a light-particle dark-matter candidate that would also solve the strong-CP problem.

### 1.3 Stellar Astrophysics & Compact Objects

- **Accretion disk** — infalling gas around a compact object, heated to emit thermal (and non-thermal) radiation.
- **AGB star (asymptotic giant branch)** — an evolved star supplying carbon, nitrogen, and s-process elements to the interstellar medium.
- **Blackbody radiation** — thermal continuum emission approximated by a star's photosphere.
- **Black dwarf** — the theoretical cold end-state of a white dwarf after ~10¹⁵ years.
- **Black hole** — a collapsed stellar remnant emitting nothing directly detectable except via accretion.
- **Blue dwarf** — a theoretical future contraction/heating stage of a dying low-mass red dwarf.
- **Boson star / gravastar / preon star** — purely theoretical compact objects serving as black-hole mimickers.
- **Bremsstrahlung** — thermal X-ray emission from decelerating free electrons in ionized/shocked gas.
- **Chandrasekhar limit (~1.4 M☉)** — the maximum mass for a white dwarf before collapse or explosion.
- **Cepheid variable** — a pulsating star used as a cosmic distance indicator.
- **Degenerate matter / electron degeneracy** — quantum pressure supporting white dwarfs against gravitational collapse.
- **Direct collapse** — formation of a black hole without a preceding supernova explosion.
- **Effective temperature (T_eff)** — a star's surface temperature, setting the shape of its emitted spectrum.
- **Fast radio burst (FRB)** — a brief, powerful radio transient, now linked to magnetars.
- **Frozen star** — either an old term for a black hole (time appears frozen at the horizon) or a hypothetical far-future cold star.
- **Hawking radiation** — negligible theoretical thermal emission from a black hole.
- **Hertzsprung–Russell (H–R) diagram** — a temperature-vs-luminosity plot mapping stellar evolution.
- **HII region** — an ionized gas region driven by hot O-star radiation.
- **Inverse-Compton scattering / Comptonization** — upscattering of low-energy photons by hot electrons, producing hard X-ray power-law spectra.
- **Iron star** — an extreme-future speculative object formed via quantum-tunneling cold fusion/fission toward iron-56.
- **Luminosity class (Morgan–Keenan system)** — a star classification by size/evolutionary state (from Ia+/0 supergiants through VII/D remnants).
- **Magnetar** — a neutron star with an extreme magnetic field (10¹⁴–10¹⁵ G), powered by field decay.
- **Main sequence** — the hydrogen-fusing life stage of a star.
- **Main-sequence lifetime** — the duration a star burns hydrogen on the main sequence; used as the "hazard clock" for a star's eventual collapse.
- **Mass gap (~2.5–5 M☉)** — the once-empty mass range between the heaviest neutron stars and lightest black holes.
- **Mass–luminosity relation** — an empirical relation used to infer stellar mass from luminosity.
- **Neutron star** — an ultra-dense stellar remnant supported by neutron degeneracy pressure.
- **Pair-instability supernova** — an explosion of the most massive Population III stars that leaves no remnant.
- **Photosphere** — a star's visible surface layer, the source of its emergent blackbody spectrum.
- **Population I / II / III stars** — stellar generations classified by metallicity and age (metal-rich/young, metal-poor/old, hypothesized metal-free first stars).
- **Protostar / young stellar object (YSO)** — a star still embedded in its natal dust.
- **Pulsar** — a rotating neutron star beaming coherent radio (or other) radiation.
- **Quark star / strange star** — a hypothetical compact object composed of deconfined quark matter.
- **Quasar / radio-loud AGN / microquasar** — luminous, accretion-powered compact-object classes.
- **Red giant / red supergiant** — evolved, cool, luminous stars radiating strongly in red/infrared light.
- **Reionization** — the cosmic epoch during which the first stars ionized the intergalactic medium.
- **Spectral class (O B A F G K M, plus L T Y, W, C, S)** — the Harvard temperature-based star classification system.
- **Spectral energy distribution (SED)** — the fractional radiation output of an astronomical object across wavelength bands.
- **Stefan–Boltzmann law (L = 4πR²σT⁴)** — the formula for a star's total power output from its radius and temperature.
- **Stellar archaeology** — reading early-universe conditions from surviving low-mass Population II stars.
- **Stellar corona** — a magnetically heated outer stellar atmosphere producing X-rays, especially in cool active stars.
- **Strange quark matter** — the hypothesized true ground state of dense matter.
- **Supermassive star / quasi-star** — a hypothesized 10⁴–10⁵ M☉ object that collapses into a massive black-hole seed.
- **Supernova remnant (SNR)** — expanding shocked debris from a supernova, radiating via synchrotron, bremsstrahlung, and line emission; also a marker of a past sterilizing event.
- **Synchrotron radiation** — non-thermal emission from relativistic electrons spiraling in magnetic fields.
- **Thorne–Żytkow object** — a hypothetical hybrid of a neutron star embedded in a red supergiant's envelope.
- **Tidal deformability** — a gravitational-wave observable constraining the neutron-star equation of state.
- **Tolman–Oppenheimer–Volkoff (TOV) limit (~2.2–2.3 M☉)** — the maximum mass for a neutron star.
- **Vegetation red edge (VRE)** — see §1.7.
- **Wien's displacement law (λ_peak ≈ 2.9×10⁶ nm·K / T)** — the formula fixing a blackbody's peak emission wavelength from its temperature.
- **White dwarf** — a degenerate stellar remnant of low/mid-mass stars, capped by the Chandrasekhar limit.
- **Wolf–Rayet star** — a massive star stripped of its hydrogen envelope, UV-dominated with broad emission lines; a core-collapse/long-GRB progenitor candidate.
- **X-ray binary** — a stellar-mass black hole or neutron-star system with an X-ray-bright accretion disk.

### 1.4 Galactic Structure & Chemical Evolution

- **[Fe/H]** — the iron-to-hydrogen abundance ratio relative to solar; the standard metallicity index.
- **[α/Fe] ratio** — a diagnostic abundance ratio recording a stellar population's star-formation tempo; high values mark a thick-disk population.
- **α-elements (alpha elements)** — oxygen, magnesium, silicon, neon; produced rapidly by core-collapse supernovae.
- **Age–metallicity relation** — the trend of metallicity rising with cosmic time across stellar generations.
- **Big Bang nucleosynthesis** — production of hydrogen, helium, and trace lithium in the early universe.
- **Chemical tagging** — a technique for reconstructing stellar birth origins from detailed abundance patterns.
- **Core-collapse supernova** — the explosion of a massive star; the dominant early source of α-elements.
- **Galactic Habitable Zone (GHZ)** — an annular region of the galactic disk balancing high metallicity (favoring planets) against low hazard rates.
- **Galactocentric radius** — distance from the galactic center; a key variable in habitability and hazard mapping.
- **Geodynamo** — the process generating a planet's protective magnetic field.
- **Hot Jupiter** — a giant planet in a very close orbit, possibly over-produced at very high metallicity.
- **Inside-out disk formation** — a galaxy-formation model in which the inner disk assembled and enriched earlier than the outer disk.
- **Iron-peak elements** — iron and nickel, produced predominantly by Type Ia supernovae.
- **Kilonova** — the electromagnetic counterpart of a neutron-star merger.
- **Metallicity gradient (radial)** — the decline of [Fe/H] with increasing distance from the galactic center (~−0.06 dex/kpc in this project's model).
- **Neutron-star merger** — an event producing r-process heavy elements and short gamma-ray bursts.
- **Nucleosynthesis** — the process of forming chemical elements inside stars or during stellar/compact-object events.
- **Planet–metallicity correlation** — the empirical trend that giant planets form more frequently around metal-rich stars.
- **Radial velocity** — the heliocentric line-of-sight velocity of a star.
- **Radiogenic heat** — heat from the decay of long-lived radioisotopes (U, Th, K-40), powering plate tectonics and the geodynamo.
- **r-process (rapid neutron capture)** — a nucleosynthesis pathway for heavy elements, dominant in neutron-star mergers.
- **s-process (slow neutron capture)** — a nucleosynthesis pathway producing roughly half the elements heavier than iron.
- **Thick disk / thin disk** — chemically distinct old (α-enhanced) and young (iron-richer) galactic disk populations.
- **Type Ia supernova** — a white-dwarf detonation; the dominant delayed source of iron-peak elements, and a cosmological standard candle.

### 1.5 Astrophysical Hazards

- **⁶⁰Fe (iron-60)** — a radioactive isotope used as geological evidence of nearby past supernovae.
- **Beaming duty factor** — the fraction of sky covered by a GRB's jet, used to down-weight beamed-hazard likelihood.
- **Collapsar** — a rapidly rotating, low-metallicity massive star that collapses to a black hole; the progenitor of long gamma-ray bursts.
- **Fluence** — the threshold radiant energy dose (~100 J/m²) defining a mass-extinction-grade GRB event.
- **Gamma-ray burst (GRB)** — an extremely luminous transient event; the dominant life-limiting astrophysical hazard in the model.
- **Kill radius / kill distance** — the lethal range of a hazard event, e.g. ~8–50 pc for a nearby supernova depending on the study.
- **Long GRB** — a gamma-ray burst lasting more than ~2 seconds, produced by collapsing massive low-metallicity stars; the dangerous, high-luminosity class.
- **Magnetar giant flare** — an extremely energetic burst from a magnetar; short-ranged but locally disruptive.
- **Ozone depletion / radiolysis** — the mechanism by which GRB or supernova radiation converts atmospheric N₂/O₂ into ozone-destroying nitrogen oxides.
- **Poisson hazard** — a statistical framing describing rare, randomly timed catastrophic events.
- **Short GRB** — a gamma-ray burst lasting less than ~2 seconds, produced by neutron-star mergers; rarer and far less lethal than long GRBs.
- **Spiral-arm passage** — a solar system's transit through a spiral arm, elevating supernova and cosmic-ray exposure.
- **Stellar superflare** — an extreme flare event; a chronic hazard, especially for planets around M dwarfs.
- **Survival probability (S)** — the hazard-aware Drake-equation term for the chance a habitable world isn't sterilized before life can arise.

### 1.6 Molecular Biology & Genetics

- **Antiparallel strands** — DNA's two strands run in opposite 5′→3′ directions.
- **B-form / A-form / Z-form DNA** — alternative helical conformations of the DNA molecule.
- **C-value enigma** — the poor correlation between genome size and organismal complexity.
- **Central dogma** — the DNA → RNA → protein flow of genetic information.
- **Chromatin / nucleosome / histone** — proteins and DNA-packaging structures that compact the genome.
- **Codon** — a nucleotide triplet specifying an amino acid or a stop signal.
- **Complementary base pairing** — the A–T (two hydrogen bonds) and G–C (three hydrogen bonds) pairing rule.
- **Degenerate genetic code** — redundancy in which multiple codons specify the same amino acid, buffering against mutation.
- **Double helix** — DNA's canonical structure.
- **Epigenetic marks** — chemical tags that alter gene expression without changing the underlying sequence.
- **Error threshold / error catastrophe (L_max ≈ 1/μ)** — Eigen's principle that genome length is capped by copying fidelity, beyond which mutations overwhelm selection.
- **Genetic firewall** — a recoded organism resistant to natural viruses due to reassigned codons.
- **GC content** — the proportion of G–C base pairs in a sequence, correlating with thermal stability.
- **Hachimoji DNA** — an eight-letter synthetic genetic system supporting Darwinian evolution.
- **Lethal mutagenesis** — an antiviral strategy of pushing a pathogen's mutation rate past its error threshold.
- **Major / minor groove** — structural helix features read by DNA-binding proteins.
- **Microproteins / smORFs (small open reading frames)** — recently discovered functional peptides shorter than the old ~50–100 amino acid minimum.
- **Neutral networks** — sparse, interconnected webs of functional sequences in sequence space that enable evolution without crossing non-functional valleys.
- **Non-canonical amino acids** — amino acids outside the standard 20, enabled by expanded genetic codes.
- **Nucleotide** — the DNA/RNA monomer, composed of a phosphate, a sugar, and a nitrogenous base.
- **Okazaki fragments** — discontinuously synthesized DNA fragments on the lagging strand during replication.
- **Open reading frame (ORF)** — a DNA stretch bounded by start/stop codons.
- **Origin of replication** — a genomic site where DNA polymerase begins copying.
- **Phosphodiester bond** — the bond linking nucleotides into the DNA backbone.
- **Purine / pyrimidine** — the two chemical families of DNA bases (double-ring: adenine, guanine; single-ring: cytosine, thymine).
- **Semiconservative replication** — DNA replication in which each original strand templates a new complementary partner.
- **Sequence space** — the combinatorial space of all possible nucleotide or amino-acid sequences.
- **Shannon information (bits)** — an information-theoretic measure; each DNA base carries 2 bits (log₂4).
- **Sugar-phosphate backbone** — the structural chain of DNA, with inherent 5′/3′ directionality.
- **Telomere / centromere** — chromosome-end caps and the segregation-organizing region of a chromosome.
- **Thermophile / extremophile** — organisms adapted to extreme conditions; thermophiles skew GC-rich.
- **Transcription / translation** — the DNA-to-RNA and RNA-to-protein steps of gene expression.
- **Unnatural base pair** — a synthetic base pair (e.g. dNaM–dTPT3) incorporated into a living organism's genome.
- **Xenonucleic acids (XNA)** — alternative-backbone nucleic acids used to probe what DNA chemistry is truly necessary for life.

### 1.7 Energy, Photosynthesis & Biosignatures

- **Anoxygenic photosynthesis** — bacteriochlorophyll-based photosynthesis using lower-energy photons, producing no oxygen.
- **Bandgap** — the semiconductor property that determines which photon energies a photovoltaic cell can convert to electricity.
- **Betavoltaics** — a technology converting beta-particle (radioactive decay) emission directly to electricity.
- **Biosignature** — a detectable sign of biological activity.
- **Chemosynthesis / chemolithotrophy** — energy harvesting from chemical redox gradients rather than light.
- **Exergy** — the useful (shortwave) energy available to drive chemistry; used to compare photosynthesis potential across star types.
- **FaRLiP (far-red light photoacclimation)** — a cyanobacterial adaptation extending photosynthesis to ~750 nm via chlorophyll d/f.
- **Great Oxidation Event** — the ~2.3-billion-year-ago atmospheric oxygenation event caused by oxygenic photosynthesis.
- **Multiphoton / photon-pooling antennae** — a hypothesized biological mechanism for combining multiple infrared photons to reach the energy of one visible photon.
- **Oxygenic photosynthesis** — chlorophyll-based, water-splitting photosynthesis that produces oxygen; requires photons above ~1.8 eV (the "red limit," ~700 nm).
- **Photon energy (E = hc/λ)** — the governing principle that photon energy, not just photon count, drives photochemistry.
- **Photosynthetic photon limit / red limit** — the minimum photon energy (~1.8 eV, ~700 nm) needed for oxygenic photosynthesis.
- **Photovoltaics** — semiconductor technology converting photon energy to electricity, tunable to a star's spectrum.
- **Radiotrophy** — the hypothesized use of melanin to derive metabolic benefit from ionizing radiation.
- **Radiovoltaics / RTG (radioisotope thermoelectric generator)** — technologies converting ionizing radiation or decay heat into power.
- **Retinal-based phototrophy / rhodopsin** — a simpler, proton-pumping light-harvesting pigment system (the "Purple Earth" hypothesis).
- **Technosignature** — a detectable sign of technology.
- **Thermoelectric conversion** — energy conversion from temperature gradients rather than photons.
- **Thermophotovoltaics** — infrared-tuned photovoltaic technology suited to cool-star/red-dwarf spectra.
- **Vegetation red edge (VRE)** — a sharp reflectance jump near 700 nm caused by chlorophyll, used as a remote biosignature.

### 1.8 Habitability & Search Frameworks

- **Abiogenesis** — the origin of life from non-living matter; a candidate step in the Great Filter.
- **Bayesian argument** — probabilistic reasoning applied to interpreting the significance of "no detections yet" in SETI.
- **Drake Equation terms (R\*, fp, ne, fl, fi, fc, L)** — the seven multiplied factors estimating the number of communicative civilizations in the galaxy (star-formation rate, fraction with planets, habitable planets per system, fraction where life/intelligence/technology arise, and broadcast lifetime).
- **foc / fpt** — proposed extensions to the Drake Equation: the fraction of habitable exoplanets with significant continents/oceans, and the fraction with long-lived plate tectonics.
- **Great Filter** — Robin Hanson's hypothesis that some step from lifeless planet to interstellar civilization is improbable enough to filter out nearly everyone.
- **Habitable zone (HZ)** — the orbital distance range around a star where a planet could sustain liquid water.
- **Narrowband radio signal** — the classic SETI search target, potentially broadened or masked by stellar space weather.
- **Plate tectonics** — a geological process tied to habitability, radiogenic heat, and long-term climate stability.
- **Rare Earth hypothesis** — the hypothesis that habitable, life-bearing planets are exceedingly rare.
- **Stellar space weather** — plasma turbulence from solar wind/stellar activity that can broaden and hide radio technosignatures.
- **Tidal locking** — a planet's rotation locked to its orbital period; a habitability concern for planets orbiting M dwarfs.
- **Zoo hypothesis / Dark Forest hypothesis** — proposed solutions to the Fermi Paradox: civilizations either deliberately avoid contact, or broadcasting is dangerous in an arms-race galaxy.

### 1.9 Kardashev-Scale & Megastructure Concepts

- **Dyson sphere** — a hypothetical megastructure enclosing a star to capture most or all of its energy output.
- **Dyson swarm / shroud** — the modern preferred architecture: a dense swarm of independent absorbing satellites, rather than a rigid shell.
- **Infrared excess** — anomalous excess infrared emission; a proposed Dyson-sphere detection signature.
- **Matrioshka brain** — nested Dyson-swarm-like shells at successively cooler temperatures; a speculative post-biological computation megastructure.
- **Microlensing** — a gravitational-lensing technique proposed to detect Dyson-sphere-like structures around black holes.
- **Two-temperature thermal model** — a model of a Dyson swarm with distinct inner/environment and outer/waste-heat temperatures.
- **Type I / II / III civilization** — the energy-harnessing tiers of the Kardashev Scale (planetary, stellar, and galactic energy budgets, respectively).
- **Type IV / Type V civilization** — speculative tiers beyond Kardashev's original three, harnessing universe- or multiverse-scale energy.

### 1.10 Data Modeling, Statistics & Software Architecture

- **Astrometry** — the precise measurement of stellar positions and motions.
- **Bolometric correction** — an adjustment converting a single-band magnitude into total (bolometric) luminosity.
- **Cell (data-modeling sense)** — a stored value bundled with its companion source, provenance tier, and confidence attributes.
- **Chunked arrays** — array storage divided into blocks for efficient partial access; used for the time-sliceable veil field.
- **Confidence (field-level)** — a [0,1] trustworthiness value attached to a single field's value from a specific source; replaces an earlier boolean `is_speculative` flag.
- **Confidence propagation / aggregation** — the method of combining per-field confidences (weakest-link input × science-backing weight, magnitude-weighted across factors) into a location's overall trust score.
- **Conflict resolution (multi-source fields)** — the rule for surfacing the highest-confidence value when multiple sources supply the same field.
- **Cross-matching** — joining or reconciling records for the same object across different catalogs.
- **Data levels (0/1/2/3)** — a classification of how processed astronomical data is, from raw pixels to science-ready catalogs.
- **De-redden** — correcting a measurement for the dimming/reddening effect of interstellar dust.
- **Detection-method selection effects** — sampling bias in confirmed-exoplanet catalogs toward certain detection techniques.
- **Disk density profile** — a double-exponential model of stellar density as a function of galactic radius and height.
- **Entity (data-modeling sense)** — a star, planet, hazard source, remnant, or cluster represented as a single record in the project's schema.
- **Entity resolution** — the process of matching and merging records that refer to the same real-world object across sources.
- **Equilibrium temperature** — a planet's theoretical temperature derived from stellar flux and albedo.
- **Extinction (dust extinction)** — the dimming/reddening of starlight by interstellar dust.
- **Factor (model concept)** — a single positive or negative habitability influence, carrying a sign, magnitude, science-backing weight, spatial profile, and temporal profile.
- **Factor contribution (log-odds term)** — the per-factor additive log-odds term combined to build the habitability field.
- **Fail-safe default** — the design principle that uncertainty is the default state; certainty must be actively earned/stated (absent confidence defaults to 0.10, Speculative).
- **Field descriptor** — the required attribute set describing any single data field in the schema.
- **Field-provenance ledger** — a long-format audit table retaining every source's value for every field, including non-surfaced ones.
- **Galactocentric coordinates** — a Cartesian (x, y, z) coordinate system centered on the galaxy, used for radius-dependent modeling.
- **Grain (data-modeling sense)** — what one record in a data store represents (e.g., "one row per resolved entity").
- **Habitability field / the veil** — the modeled continuous probability surface of habitability over galactic space and time.
- **Heliocentric distance / coordinates** — distance or position measured from the Sun, as opposed to the galactic center.
- **ICRS (International Celestial Reference System)** — the celestial coordinate reference frame for right ascension/declination.
- **Insolation** — the stellar flux received by a planet, typically expressed relative to Earth's.
- **k-d tree / octree** — spatial-indexing tree structures for fast nearest-neighbor or radius queries.
- **Layered data storage (ingest / canonical / derived)** — the project's three-tier data architecture: staging by source, canonical records by entity, and derived products by space-time.
- **Log-odds / logit** — the additive mathematical space in which independent habitability factors are combined before conversion back to a probability.
- **Log-odds factor model** — the project's core modeling approach: factors combine additively in log-odds space, then pass through a logistic function to yield a valid [0,1] probability.
- **Logistic transform / expit** — the function mapping combined log-odds back to a probability; the inverse of the logit.
- **Magnitude-limited (survey)** — a survey complete only down to a given brightness, not out to a given distance or volume.
- **Optimum band (∩-shaped factor)** — a habitability factor that is unfavorable at both extremes and favorable in the middle (e.g., metallicity).
- **Parallax** — an astrometric angle used to derive a star's distance (distance in parsecs ≈ 1000 / parallax in mas).
- **Point hazard / point source** — a discrete, spatially localized hazard entity, as opposed to a smooth statistical field.
- **Population synthesis** — statistically generating realistic synthetic stellar populations to fill regions of the galaxy without direct observations.
- **Proper motion** — a star's apparent angular motion across the sky over time.
- **Provenance** — the traceable origin of a data value, tied to a source and a confidence level; "the difference between an instrument and a pretty lie."
- **Provenance tier** — one of five canonical trust categories (Measured/Direct, Inferred, Statistical-only, Modeled, Speculative) governing a field's default confidence.
- **Spatial index** — a data structure enabling fast proximity/lookup queries against entity positions.
- **Spatial profile f(d) / temporal profile g(t)** — the shared functions describing how a factor's effect varies over distance and over time, respectively.
- **Truncated inverse-square falloff** — the spatial profile shape used for point hazards: near-1 close to the source, decaying with distance, with a hard cutoff beyond a threshold.
- **Vectorization** — the requirement that field math be expressed as array operations (e.g., NumPy) rather than per-cell loops, for performance.
- **Volume-complete sample** — a survey or catalog complete out to a fixed distance, regardless of brightness (contrasted with magnitude-limited).
- **Weakest-link (input confidence)** — the propagation rule that uses the least-confident contributing input when combining confidences.

### 2.4 Named Equations, Laws & Scales

## 3. Code Libraries, Tools & Data Formats

Top-level libraries, tools, and formats only — not individual function or method calls.

- **ADQL (Astronomical Data Query Language)** — the SQL-like query language used to query the Gaia Archive and similar services.
- **astropy** — Python astronomy library; used specifically for coordinate transforms (RA/Dec/distance → galactocentric Cartesian) and unit handling.
- **astroquery** — Python package for querying astronomical archives (Gaia, SIMBAD, VizieR); deferred past Phase 0.
- **CSV** — plain-text tabular format used for AT-HYG, NASA Exoplanet Archive, and other bulk catalog downloads.
- **Dask** — an out-of-core dataframe library for billion-row (Gaia-scale) data, deferred to Phase 3.
- **deck.gl** — a WebGL-based geospatial/3D visualization library; a candidate Phase 1 rendering stack.
- **DuckDB** — an embedded columnar database that reads Parquet files directly; the planned next step for querying at scale.
- **FITS** — an astronomical binary data format offered for bulk catalog downloads.
- **GitHub** — the hosting platform for the HYG database repository and the project's own repository.
- **HTML** — the output format for the self-contained interactive Plotly deliverable.
- **Markdown** — the document format used for all project artifacts and research documents.
- **matplotlib** — a fast, dependency-light Python plotting library used for static validation renders (heatmaps, contour plots, 3D surfaces).
- **NetCDF** — a chunked, time-sliceable array storage format, considered alongside Zarr for the derived veil-field store.
- **NumPy** — the Python numerical array library underlying the veil-field representation and vectorized factor math.
- **pandas** — the Python data-analysis library used for catalog ingestion, cleaning, and joining.
- **Parquet** — the columnar file format used for the staging, entities, and field_provenance data stores.
- **pip** — the Python package installer used to install and pin project dependencies.
- **Plotly** — the primary interactive rendering library for Phase 0 (rotatable 3D surfaces, hover tooltips, time slider).
- **Python** — the programming language chosen for Phase 0 and the data pipeline.
- **SciPy** — a Python scientific-computing library; provides the `cKDTree` spatial-index implementation and the numerically stable logistic function (`expit`).
- **Three.js** — a WebGL 3D JavaScript library; a candidate Phase 1 rendering stack, offering maximum control.
- **Vaex** — an out-of-core dataframe library, an alternative to Dask for billion-row datasets.
- **venv** — the Python virtual-environment tool used for isolated, reproducible development environments.
- **VOTable** — a heterogeneous astronomical tabular data format alongside CSV.
- **WebGL** — the browser 3D graphics API underlying Three.js and deck.gl; the Phase 1 rendering target.
- **Zarr** — a chunked N-dimensional array storage format; the planned successor to in-memory NumPy arrays for the veil field once it outgrows RAM.
